一个算sum_k,sum_b的函数

算var_sum的函数来了

算k，b的过程如下
控制一个精度，当算出来的var_sum满足一定条件后，在里面不断改变k,b的值，知道不满足条件，输出最后的k,b

l =                                  # 小姐姐请自行调参
while l > :                          # 小姐姐请自行调参
    sum_k = float(sta(file, k, b, length)[0])
    sum_b = float(sta(file, k, b, length)[1])
    temp_k = k - x * sum_k
    temp_b = b - x * sum_b
    k = (temp_k)
    b = (temp_b)
    l = float( var(file,k,b,length) )
else:
    print('%f %f' % (k, b))
    writelist_k = str(k)
    writelist_b = str(b)
    writelist = 'k:' + writelist_k + '\n' + 'b:' + writelist_b
    with open('.txt','w') as fi:     # 记得输入文件名                    <-覆盖写模式
        fi.write(writelist)

print(‘––END––’)
